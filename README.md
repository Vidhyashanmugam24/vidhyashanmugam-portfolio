# Vidhya Shanmugam | Professional React Portfolio

A modern, highly professional full-stack developer portfolio built with React.js, Vite, and custom CSS styling. Features a premium dark/light mode toggle with custom design tokens, responsive layouts, and structured sections showcasing professional journey, skills, and credentials.

## 🚀 Features

- **Premium Light & Dark Mode Toggle**: Responsive dark/light theme switching with smooth transitions and CSS-custom variables.
- **Detailed Pages**:
  - **Home**: Introducing Vidhya Shanmugam, professional background, social profiles, and links.
  - **Skills**: Structured interactive cards presenting 14 key technical and professional categories.
  - **Education**: Detailed timelines displaying academic achievements separately (MCA & BCA).
  - **Experience**: Structured professional history with role descriptions, locations (Bangalore, Ontario), project details, and comprehensive technical bullet points.
  - **Accomplishments**: Catalog of professional certifications with custom card layout.

## 🛠️ Technology Stack

- **Frontend Core**: React.js, JavaScript (ES6+), HTML5, CSS3
- **Build Tool**: Vite (configured with React automatic JSX runtime)
- **Styling**: Vanilla CSS with custom utility variables & Bootstrap integration
- **Icons**: React Icons (`react-icons`)
- **Routing**: React Router DOM (`react-router-dom`)

## 💻 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository or navigate to the project directory:
   ```bash
   cd professional-react-portfolio
   ```

2. Install the project dependencies:
   ```bash
   npm install
   ```

### Development Server

Run the development server locally:
```bash
npm run dev
```
The application will start and run by default at `http://localhost:5174/`.

### Production Build

To build the production bundle:
```bash
npm run build
```
The compiled build output will be placed in the `dist/` directory, optimized and ready for deployment.
